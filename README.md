# Hostel Room Allocation System

A simple Python-based console application to manage hostel room assignments efficiently.  
This system keeps track of students, available rooms, room allocations, and allows easy updates.

## Features
✔ Add new students  
✔ Allocate a room to any registered student  
✔ Prevent allocation of occupied rooms  
✔ View all room allocations  
✔ Remove a student (automatically frees their room)  
✔ Display available rooms  
✔ Menu-driven program for easy use  

## Tech Used
- **Python 3**
- Built using basic concepts:
  - Dictionaries
  - Functions
  - Loops
  - Conditional statements


## How It Works
The program maintains two main dictionaries:

### `hostel_rooms`
Stores room numbers and their current occupant roll number.

### `students`
Stores student details like name and assigned room.

Users can interact with the system through a menu:
1. Add Student  
2. Allocate Room  
3. View Allocations  
4. Remove Student  
5. Show Available Rooms  
6. Exit
